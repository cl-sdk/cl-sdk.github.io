# cl sdk

[![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](http://unlicense.org/)

> I'll never write these functions again.

cl-sdk will, hopefully, solve the problem of writing all those packages again.

The goal is to provide a foundation for common lisp development with a permissive license.

It provide a collection of repositories with some useful packages.

Benefits for supporting this initiative:

- Never write those packages again.

- Use well-know and tested packages.

- Permissive license
